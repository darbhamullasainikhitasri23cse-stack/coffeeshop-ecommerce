# CoffeeShop E-Commerce Website

A full-stack coffee shop e-commerce application built using HTML, CSS, JavaScript, Node.js, and Express.

## Features
- Product listing
- Contact form
- Backend server using Node.js
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express

## How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `node server.js`
4. Open http://localhost:3000
